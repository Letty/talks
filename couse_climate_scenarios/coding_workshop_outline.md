### Coding
* Variables and Data structure

    - let
    - Simple Variables
        + `let a = 'Hello';`
        + `let b = 23847628;`
        + `let c = 2.342;`
    - Arrays
        + `let d = [1,2,45,21,52,66,89,123];`
        + `let e = [1,'hello', 234.32,'foo'];`
    - Objects
        + `let f = {'name': 'John', 'age': 42} `
* Control Flows
    - Conditions
        + if..else
            ```javascript
            if (condition === true){
                doSomething();
            }else{ // condition false
                doAnotherThing();
            }
            ```
       
        + switch
            ```javascript
            switch(expression):
                case value1:
                    do();
                    break;
                case value2:
                    doTwo();
                    break;
                default:
                    doDefault();
            ```
    - Loops
        + for
        + forEach
        + while / do while
